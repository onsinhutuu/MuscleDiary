# <MUSCLE DIARY>

## サイト概要
トレーニングの種目、回数、その日の体調や意気込みなどを日記のように記録できる

### サイトテーマ
日々のトレーニング内容（主にウエイトトレーニング）を記録できるサイト

### テーマを選んだ理由
自分の趣味が筋トレであり、自分が本当に成長しているのか数値を記録したいが良いサイト、アプリがない為自分で作ろうと思ったから

### ターゲットユーザ
筋トレをしていて、自分の成長を記録したい人

### 主な利用シーン
筋トレ後、筋トレセット間休憩中
